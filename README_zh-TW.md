## 簡介

[Note2Link](https://note2link.com) 是一款專為筆記應用及海外雲存儲服務提供加速及同步服務的多平台桌面應用程序。它能夠顯著提升您在使用各類筆記軟件和雲存儲服務時的連接速度和穩定性，為知識工作者和研究人員提供更加流暢的使用體驗。

### 主要特性

- **線上工具加速**：優化連接，提高Gmail、Notion、Roam Research、RemNote、Tana、Google Keep等生產力和知識管理工具的訪問速度
- **雲存儲服務同步**：支持GitHub、Google Drive、Microsoft OneDrive、Dropbox、MEGA等多種雲存儲服務的同步加速
- **多平台支持**：適用於Windows、macOS和Linux系統
- **多瀏覽器兼容**：支持Chrome、Firefox、Edge、Safari等主流瀏覽器
- **系統托盤集成**：最小化到系統托盤，隨時可用但不干擾工作流
- **多語言支持**：提供多種語言界面，滿足全球用戶需求
- **智能代理設置**：自動配置系統代理，無需複雜設置
- **自動更新**：內置更新檢測系統，確保您始終使用最新版本

## 安裝

從[官方網站](https://note2link.com)下載適合您系統的安裝包。

### Windows

- 下載`.exe`安裝文件
- 雙擊運行安裝程序，按照提示完成安裝

### macOS

- 下載`.dmg`文件
- 打開DMG文件，將應用拖到應用程序文件夾
- 首次運行時，您可能需要在系統偏好設置中允許來自未知開發者的應用

### Linux

- 下載`.AppImage`或適合您發行版的包
- 使其可執行: `chmod +x Note2Link-Desktop.AppImage`
- 直接運行應用程序: `./Note2Link-Desktop.AppImage`

## 使用方法

1. 啟動應用程序後，在加速選項卡頁面通過加速開關按鈕一鍵啟用服務。新用戶可以點擊"開始試用"免費體驗（有時間限制），或查看[訂閱計劃](https://note2link.com/zh-TW/index#pricing)獲取完整功能（付費用戶享有無限制使用及IPLC高速節點自動切換）
2. 在設置中配置應用行為，如自動啟動、界面主題等
3. 根據需要設置代理端口，查看[端口設置說明](https://github.com/Note2Link/Note2Link/blob/main/port_setting_zh-TW.md)獲取詳細指南
4. 使用應用內幫助獲取詳細的使用指南

## 配置

[Note2Link](https://note2link.com) 提供了多種配置選項：

- **基本設置**：自動啟動、語言選擇、界面主題
- **端口設置**：配置系統代理和瀏覽器代理端口
- **更新設置**：自動檢查更新、更新通知
